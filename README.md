# wiringpi-test

It is a package that helps test wiringpi and target board.

### Install & Usage

install
```
$ sudo add-apt-repository ppa:how2flow/ppa
$ sudo apt update
$ sudo apt install wiringpi-test
```

usage
```
$ wiringpi-test
```

### Requirements

```
- libwiringpi2
- libwiringpi-dev
- wiringpi
```

### Supports

```
- odroid
- raspberry pi
```
